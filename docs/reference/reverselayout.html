<reverselayout>

A layout for arranging siblings vertically or horizontally in reverse order.

Extends layout

<para> reverselayout is just like <simplelayout> , except that the order of the views it keeps is reversed (bottom to top, or right to left). </para> <para> reverselayout extends <layout> , and therefore it is responsible for arranging all of the subviews for the view that it is attached to. The attributes of a view that reverselayout effects are the x and y values. </para> 
<canvas height="60" >
  <include />
  <view bgcolor="yellow">
    <reverselayout axis="x" spacing="10"/>
    <text text="one"/>
    <text text="two"/>
    <text text="three"/>
  </view>
</canvas>
                     
<canvas height="60">
  <include />
  <view bgcolor="yellow">
    <reverselayout axis="x" end="false" spacing="10"/>
    <text text="one"/>
    <text text="two"/>
    <text text="three"/>
  </view>
</canvas>

Attributes

NameTypeDefaultDescription
axisstringyThe axis in which this layout operates. One of 'x' or 'y'.
endexpressiontrueWhen true (the default), then the layout will push the subviews to the right (or bottom) of the containing view
insetexpression0A pixel amount to inset the last view controlled by the layout
sizeAxisstringheight
spacingexpression0A pixel amount to use between each view in the layout

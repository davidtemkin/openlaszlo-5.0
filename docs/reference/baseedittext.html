<baseedittext>

Extends baseformitem

This component provides the look and feel for user editable text, data binding, and focus events.

Attributes

NameTypeDefaultDescription
fgcolorexpressionSetting the fgcolor of text has the unfortunate effect of setting the hilite color as well, so it is disabled here.
fontsizeexpression13Font size of the text.
heightexpression20The height of the edit text box.
maxlengthnumbernullThe maximum number of chars that can be entered. Default is unlimited.
multilineexpressionfalseSet to true for multiple lines.
passwordexpressionfalseSet to true for a password field. Text will appear as ****. This attribute cannot be changed at runtime.
patternstringnullAn expression to restrict the set of characters that are allowed to be entered. For example, [0-9]*, [012]*, [a-zA-Z]*, [0-9A-Z]*. You can restrict the number of characters using maxlength. Default is any character.
texthtmlThe text that appears in the component. Text and value are the same.
valueexpressionnullThe text that appears in the component. Text and value are the same.
widthexpression106The width of the edit text box.

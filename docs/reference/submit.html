<submit>

associates a form with a dataset

Extends node

Used with form or baseform , the submit tag associates the form with its dataset. See form tag for example code.

See also: <tags>form baseform dataset</tags>

Associates a form with a dataset.

Attributes

NameTypeDefaultDescription
datasetexpressionnullThe dataset that specifies the request url, which will also hold the result of the query.
errorstringError from most recent response or empty string; cleared to empty string each time a request on the dataset is made.
statusstringokRead-only status of data request. Possible values are 'ok', 'requesting', and 'error'.

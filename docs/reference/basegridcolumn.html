<basegridcolumn>

An abstract column in a grid.

Extends basecomponent

basegridcolumn can be used in basegrid s or grid s to customize the look of both the column header and the row contents for that column. In cases where a standard column header look is desired, but the contents may vary, use gridcolumn .

The placement of "header" can be used to place nodes in the header of the grid, and the default placement for the class will place things in the row. See the example in basegrid .

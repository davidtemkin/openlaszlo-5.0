<mediadevice>

The base class for media devices.

Extends node

As of OpenLaszlo release 4.0, subclasses of this class are <microphone> and <camera> .

Attributes

NameTypeDefaultDescription
activeexpressionfalseFlag that tracks if there's device activity (audio sound, video motion).
allowedexpressionfalseTracks whether the user has allowed device access. True after user has accepted security dialog, false if the user denies access with security dialog.
capturingexpressiontrueBoolean flag that controls if the media device is currently capturing input.
debugexpressionfalseset to true for verbose debugging messages
deviceindexexpressionnullZero-based index of the device, or null for user's default device. Default is null, which selects the default device. When the default device is initialized, deviceindex is updated to its actual index.
devicenamestringName of the device.

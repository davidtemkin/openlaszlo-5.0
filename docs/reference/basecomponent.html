Extends LzView

An abstract base class that handles focus, the text attribute, and style.

an abstract class which all components subclass

Attributes

NameTypeDefaultDescription
doesenterbooleanfalseif set to true, the component manager will call this component with doEnterDown and doEnterUp when the enter key goes up or down if it is focussed.
enabledexpressiontruewhen false the component should appear as its disabled state and not respond to user events. The component is automatically set to clickable when enabled, and not clickable when disabled.
focusablebooleantruewhen true, the component can be focused.
hasdefaultexpressionfalseset to true when this component will receive enter key events
isdefaultexpressionIf set to true, this component will be the default (and will therefore receive the enter key) if it is nearest to the focussed view, and if that focussed view does not have doesenter set to true. "Nearness" is defined to mean the closest common parent with the focussed view that is not excluded from the mode, and which is visible onscreen.
styleexpressionnullif null, the style is set to the inherited style from the parent component, if no parent component the style will be the canvas 'defaultstyle' attribute
styleableexpressiontruewhether style should be applied to this component
texthtmlthe label or title of the component: instead of using this attribute text may instead be placed withing the tag, for example: <button>OK</button>

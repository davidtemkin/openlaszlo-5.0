<basefocusview>

an base class for the focus indicator

Extends LzView

A base class that receives events from the focus manager, automatically showing, hiding and sizing to the focused view. See focusoverlay for a concrete example.

Attributes

NameTypeDefaultDescription
activeexpressionfalsetrue during keyboard navigation, when this view is visible
durationexpression400the duration of the animation from one focused view to the next
targetexpressionnulla reference to the view that is currently focused

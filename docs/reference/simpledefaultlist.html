<simpledefaultlist>

Extends list

A list class that handles a default value. Input and output for the list is defined to be in this format: <list> <item></item> <item></item> </list> Currently only works with dataoption="lazy" or "pooling"

Attributes

NameTypeDefaultDescription
autoselectdefaultexpressiontrueSet to true if the default item should be selected whenever data is received
defaultitemexpressionnullLzDataElement of the default node null means the list has no default

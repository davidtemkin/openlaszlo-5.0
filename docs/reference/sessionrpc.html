<sessionrpc>

SessionRPC object

Extends rpc

A class to get an http session object and invoke its methods. Remote method signatures are:

  • <dfn>getAttribute(name)</dfn> : returns object, array, or primitive type.
  • <dfn>getAttributeNames()</dfn> : returns array of attribute names.
  • <dfn>getId()</dfn> : returns remote session id
  • <dfn>getMaxInactiveInterval()</dfn> : returns max inactive interval for session.
  • <dfn>invalidate()</dfn> : invalidate remote session.
  • <dfn>isNew()</dfn> : returns true if the client does not yet know about the session or if the client chooses not to join the session.
  • <dfn>removeAttribute(name)</dfn> : remove attribute named name; returns void.
  • <dfn>setAttribute(name,value)</dfn> : set attribute named name with value; returns void.
  • <dfn>setMaxInactiveInterval(interval)</dfn> : set max inactive interval for session.

See javax.servlet.http.HttpSession in the Java Servlet API documentation for details.

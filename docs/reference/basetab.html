<basetab>

The base tab class.

Extends baselistitem

<para>This class is to be used only in conjunction with basetabs and basetabapane classes. You can pass the name of this class in the 'tabclass' attribute of basetabpane, tabpane, basetabs, and tabs classes. Basetabs uses this class as its default 'tabclass'.</para> 
<canvas height="75">
  <include />

  <class name="aquatab" extends="basetab" 
         width="40" height="20" bgcolor="aqua">
    <text>tab</text>
  </class>
  <class name="greentab" extends="basetab" 
         width="40" height="20" bgcolor="green">
    <text>tab</text>
  </class>

  <basetabs x="10" y="10" tabclass="aquatab">
    <basetabpane>
      <text>one</text>
    </basetabpane>
    <basetabpane tabclass="greentab">
      <text>two</text>
    </basetabpane>
    <basetabpane >
      <text>three</text>
    </basetabpane>
  </basetabs>
</canvas>

See Also:

Attributes

NameTypeDefaultDescription
tabpaneexpressionnullThe tabpane associated with this tab.
texttextthis.tabpane.textThe name of this tab. This is constrained to the tabpane's text.

<tooltip>

A floating "tooltip" for a view.

Extends node

The tooltip allows you to show a floating "tooltip" on any view. To use it you make it a child of a view. The parent view has to be clickable for the tooltip to work. Setting clickable to false will disable the tooltip. 

          <canvas height="60" layout="axis:y; spacing: 10">
            <text>Mouse over the button to see the tip</text>
            <button text="Button, button">
              <tooltip>Click or mouse out to hide tip</tooltip>
            </button>
          </canvas>
✎︎ Edit

The appearance of the tooltip is determined by the canvas.tooltipview element

Attributes

NameTypeDefaultDescription
texttext The text that will be displayed by the tooltip. If the text is null or empty, no tooltip will be displayed.
tipalignstringleftWhether the tooltip is horizontally aligned by it's left edge (default) or right edge relative to the mouse position.
tipvalignstringdefaultWhether the tooltip shows up above or below the mouse position. Default will position heuristically to keep the tooltip visible on the canvas.
yoffsetexpression25The vertical offset of tooltip from the mouse position.

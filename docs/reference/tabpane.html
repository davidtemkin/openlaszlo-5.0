<tabpane>

A tabpane houses content to be shown within a tabs component when its correlated tab is selected.

Extends basetabpane

tabpane is the visual container for content in . The difference between this class and is that tabpane has a background that is set by whatever the default style is.

Note that the content of tabpane is initstage="late" . You can override this by explicitly setting your initstage preference on tabpane. 


<tabpane initstage="normal">
  <!-- ...content... -->
  </tabpane>
</tabpane>

For every tabpane declared, a corresponding is created. 

                    
<canvas height="150" >
  <include />

  <simplelayout axis="y" spacing="5"/>
  <text x="10" multiline="true" width="200">
      First two are tabpanes, the last one is a basetabpane
  </text>
  <basetabs x="10" tabclass="tab" bgcolor="green">
    <tabpane>
      <text>one</text>
    </tabpane>
    <tabpane>
      <text>two</text>
    </tabpane>
    <basetabpane>
      <text>three</text>
    </basetabpane>
  </basetabs>
</canvas>

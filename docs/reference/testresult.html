<TestResult>

Extends DebugObject

A TestResult accumulates the results of a test run: the total number of tests run, the number of failures, and the number of errors. A TestResult is automatically created by a TestSuite and included as its first child view so the results of the test suite will be displayed. See the documentation for for an example of this tag.

<splash view>

A view element within the splash element positions a resource on the canvas while the application is loading.

A view tag within a splash element defines a view that is used to present a resource while the application is loading.

Only a few attributes of the standard view element can be used on this view, because it is displayed before most of the system is present.

See the splash documentation for examples.

Attributes

NameTypeDefaultDescription
centerboolean"false"If true, this view is centered within the canvas. This overrides the x and y attributes.
rationumberIf this attribute is present, the resource is synchronized to loading progress. Its value should be either a number between 0.0 and 1.0, or a percentage between 0% and 100% (inclusive). If the resource is a multiframe resource, the value is the proportion of the frames to play during the preload phase of application load; the remainder is played during the initialization phase.
xnumber"0"x position of this view relative to the canvas
ynumber"0"y position of this view relative to the canvas

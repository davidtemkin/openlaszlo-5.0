<include>

Includes code from other files.

The include tag is used to include source code contained in another file. See the Guide for a complete discussion. Also see the <library> tag.

Attributes

NameTypeDefaultDescription
hrefxsd:anyURIA reference to a target file whose content is included in the application that contains this include element.
type"text" | "xml""xml"If this attribute has the value text, the content of the target file is included as text. Otherwise, it is read as XML.

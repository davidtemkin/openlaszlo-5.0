<hbox>

A horizontal box

Extends LzView

The hbox provides a convenient way to organize a simple horizontal layout. 

            <canvas height="30" >
                <hbox spacing="10">
                    <handler name="onclick">
                        this.animate('spacing', this.spacing == 10 ? 0 : 10, 1000);
                    </handler>
                    <text>click</text>
                    <text>to</text>
                    <text>animate</text>
                    <text>spacing</text>
                </hbox>
            </canvas>

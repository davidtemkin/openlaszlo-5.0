LZX Reference

<ulink>

Extends LzView

A class for URL links. This differs from the use of the XHTML <a> attribute, e.g in '<text><a >OpenLaszlo.org</a>lt;/text>', in these ways: - There is a hover effect. (This defaults to underline.) - There is a hover color. (This defaults to blue.) - The link content and target can be databound. 
      <ulink >OpenLaszlo.org</ulink>
See ulink-test.lzx for additional examples.

Attributes

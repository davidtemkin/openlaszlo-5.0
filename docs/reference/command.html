<command>

An element that sends events in response to keyboard input.

Extends node

            <canvas height="130" debug="true">
            <command onselect="Debug.debug('the enter key was pressed');" 
                        key="['Enter']" />
            
            </canvas>

Attributes

NameTypeDefaultDescription
activebooleantrue true if the command is currently active. The default is true .
keyexpressionWTF? See LPP-4982, LPP-7814 Necessary to add "key" to the schema

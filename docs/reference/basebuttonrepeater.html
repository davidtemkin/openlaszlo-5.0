<basebuttonrepeater>

a basebutton that sends an event if the mouse is held down.

Extends basebutton

This class is a simple extension of basebutton . If the user holds the mouse button down over the button (for more than a half second), then an event <event>onmousestilldown</event> will be sent. The <event>onmousestilldown</event> event will continue to be sent every 50 milliseconds until the mouse button is released.

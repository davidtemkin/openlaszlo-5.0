LzHTTPDataProvider

DataProvider which implements HTTP request transport

Extends LzDataProvider

<class>lz.HTTPDataProvider</class> implements the DataProvider interface, to support HTTP data requests.

Requests are submitted using the <method>doRequest</method> , passing a request object of type <class>lz.HTTPDataRequest</class> . And instance of <class>lz.HTTPDataProvider</class> is the default data provider for the runtime, and is what is used by dataset to load data.

<event>onstatus</event> : Sent to a datarequest to update the status of the request.
